# Personal Portfolio

This repository contains the source code for my personal web portfolio.

The website showcases my projects, development journey, and ways to get in touch with me. It was designed to be clean, modern and responsive, focusing on usability and clear presentation of my work.

## Live Website

You can visit the portfolio here:

🔗 https://jessicayve.github.io/portfolio

## Sections

The website includes:

- **About Me** – A short introduction about me and my work
- **Tech Stack** – Technologies I work with
- **Projects** – A selection of projects I have built
- **Contact** – Ways to reach me

## Features

- Responsive design for desktop and mobile
- Clean and modern UI
- Project cards with hover effects
- Smooth navigation between sections
